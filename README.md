# toto-atividades
Códigos para a disciplina de Algoritmos e Estruturas de Dados I (AED I) 2026.
Inclui resoluções de exercícios propostos em aula e do LeetCode.

Se não ouver main ou main.exe, use o comando 'make' no terminal.

Selecionar problema: cd LeetCode/lc-[número do problema].

Executar: ./main (Linux) ou ./main.exe (Windows).
